# Security Policy

## Supported Versions
This project currently supports the latest commit on the default branch.

## Reporting a Vulnerability
Please do not open public issues for security-sensitive reports.

Use one of the following channels:
- Private security advisory in GitHub
- Direct contact with the maintainer

Include as much detail as possible:
- Affected endpoint and request data
- Reproduction steps
- Potential impact

We will acknowledge reports and work on a fix as quickly as possible.
